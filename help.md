1. Теги для создания таблиц в Latex (24.03.2026)

%===m>lib_path|macroblock  

lib_path|macroblock - получаемые методом parse_start_tag и используемые get_table_latex
lib_path - имя поля "Name" библиотечного пути JSON файла функционального блока (например, 'ДЗ (LD)_1')
macroblock - имя поля "Name" макроблока, из которого должны быть получены уставки (например, 'ДЗ-ФФ первая ступень (LN)_1')

Варианты macroblock: "Имя макроблока" / general / - / general+-
"Имя макроблока" - выводятся уставки для конкретно заданного макроблока
"general" - выводятся "Общие" уставки (уставки ФБ)
"-" - выводятся уставки всех макроблоков
"general+-" - выводятся "Общие" уставки (уставки ФБ) + уставки всех макроблоков
